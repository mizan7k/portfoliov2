# Portfolio V2

A modern, high-performance portfolio website built with React, Vite, and Tailwind CSS.

## 🚀 Deployment to GitHub Pages

This project is pre-configured for deployment to GitHub Pages.

### Prerequisites

1.  Create a new repository on GitHub named `portfoliov2`.
2.  Push this code to your repository.

### Deployment Steps

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Deploy**:
    Run the following command to build the project and push the `dist` folder to the `gh-pages` branch:
    ```bash
    npm run deploy
    ```

3.  **Configure GitHub Settings**:
    - Go to your repository on GitHub.
    - Click on **Settings** > **Pages**.
    - Under **Build and deployment** > **Branch**, select `gh-pages` and the `/ (root)` folder.
    - Click **Save**.

Your site will be live at `https://mizan7k.github.io/portfoliov2/` within a few minutes!

## 🛠️ Tech Stack

- **React 19**
- **Vite**
- **Tailwind CSS**
- **Motion** (for animations)
- **Lucide React** (for icons)
- **Formspree** (for contact form)
